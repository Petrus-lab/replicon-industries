Replicon Local Runner - Total Control Edition
============================================

Features:
---------
- Shutdown and restart the backend or UI server directly from your browser interface (output feedback included)
- All endpoints: /shutdown, /restart, /restart-ui
- Shutdown-all.bat and PowerShell scripts for automation
- PM2-ready for zero-downtime backend restarts

How To Use:
-----------
1. Install dependencies and PM2 (see pm2-setup.txt)
2. Use PM2 to run your backend (see pm2-setup.txt)
3. Run shutdown-all.bat for safe, automated shutdowns
4. Use the browser UI for one-click control (Shutdown/Restart backend and UI server)
5. All new features keep your UI dark mode and style
